Chose a data summary type.

  • None: No data summary will be performed. Raw data are returned.
  • Temporal: Data is summarized over time at each grid point. The result is a geospatial layer.
  • Spatial: Data is summarized over the seach area (select grid by choice). The result is a time series.

Choose the type of temporal summary to perform.

  • Maximum: Returns the maximum data value over the time period at each grid point.
  • Minimum: Returns the minimum data value over the time period at each grid point.
  • Mean: Returns the average of the data values over the time period at each grid point.

Choose the type of spatial summary to perform.

  • Maximum: Returns the maximum data value over all grid points at each time step.
  • Minimum: Returns the minimum data value over the grid points at each time step.
  • Mean: Returns the average of the data values over all grid points at each time step.
