What is the Summary of the Day (SOD) Daily Lister and Graph?

An interactive web-based tool for visualization of Temperature and Precipitation data.

What dataset is being used?

This tool works with daily COOP (Cooperative Observers) station data.

More on COOP data

What can you do with this tool?

Generate Temperature and Precipitation Time Series

What is the Summary of the Day (SOD) Monthly Data Lister?

An interactive web-based tool that generates daily summaries over a user chosen month/year in table form.

What datasets can be used?

This tool works with daily COOP (Cooperative Observers) station data.

More on COOP data

What can you do with this tool?

Generate daily summaries over a month for:

Temperatures (Max/Min)
Precipitation (Rain/Snow)
Evaporation
Wind Movement

What is the Summary of the Day (SOD) Monthly Summaries?

An interactive web-based tool that generates monthly summaries over a user chosen year range in table form.

What datasets can be used?

This tool works with daily COOP (Cooperative Observers) station data.

More on COOP data

What can you do with this tool?

Generate monthly summary tables for elements:

Temperatures (Max/Min/Mean/Range)
Precipitation (Rain/Snow)
Degree Days
Evaporation
Wind Movement

Available Statistics:

Maximum/Minimum
Mean
Standard Deviation
Number of Days
Range
Sum

Threshold setting available

Look at Departure from Average

Run Frequency Analysis (Pearson III/ Generalized Extreme Value)
