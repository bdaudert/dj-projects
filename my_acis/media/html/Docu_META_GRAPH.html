What is the Metadata Graphics Tool?

A tool to view availability of daily data for the period of record of a station in the COOP network.
Graphs for the five most common climate elements are generated.

  • Maximum Daily Temperature
  • Minimum Daily Temperature
  • Daily Precipitation
  • Daily Snowfall
  • Total Snowdepth

What dataset are available?

This tool works returns metadata of COOP stations.
