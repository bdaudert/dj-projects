PLOT OPTIONS
Several plot options are available.
  • Image size: Choose your image size from the drop down menue.
  • Show major grid: Dashed grid lines will be places at a step size of 1/5(data_max - data_min)
  • Show minor grid: Dashed gridlines will be placed at a step size of 1/10(data_max - data_min).
  • Connector line: Choose "Yes" if you would like your data points connected by a line.
  • Connector line width: Width of the line connecting that data points. 10 is the maximum line width allowed. Choosing 0 is the same as choosing no connector line.
  • Markers: Choose yes if you would like to see a marker at each data point.
  • Marker type: Choose a marker type from the drop down menue.
  • Vertical axis min: Set the y-axis minimum manually.
  • Vertical axis max: Set the y-axis maximum manually.
