Elements
Choose the climate elements.
Input should be a comma separarted list of element abbreviations.
Following climate elements area available.
  • Maximum Temperature (maxt)
  • Minimum Temperature (mint)
  • Average Temperature (avgt)
  • Observation Time Temperature (obst)
  • Precipitation (pcpn)
  • Snowfall (snow)
  • Snowdepth (snwd)
  • Degree Days (Heating/Cooling/Growing) (hdd,cdd,hdd)
Base Temperatures for degree days can be specfied through the element abbreviation.
Example: Heating degree days with base temperature 60F has element abbreviation hdd60.
