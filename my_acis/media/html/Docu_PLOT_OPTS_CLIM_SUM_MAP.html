PLOT OPTIONS
Several plot options are available.
  • Image size: Choose your image size from the drop down menue.
  • Number of Levels: Number of levels of the contour map. Please select a number between 1 and 20
  • Color Map: The complete set of matplotlip color maps is available. Please consult the link below the Color Map field for a complete list.
  • Interpolation: Interpolation method. Options are:
    • None: No interpolation is performed
    • Cspline: Cspline algorithm
  • Projection: Projection method. Options are:
    • Lambert Conformal Conic (LCC): LCC explained
