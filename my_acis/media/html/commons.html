How to use this tool

About this tool

What is Westmap?

An interactive web-based interface that can be used to understand climate variability.

What dataset is being used?

The initial fundamental data is monthly PRISM temperature (maximum, mean, minimum) and precipitation at 4km resolution. Period of record is 1895-present. Currently, WestMap supports gridded PRISM climate data for the Western United States.

What can you do with this tool?

Generate Maps

  • Variety of subregion options
  • Regional aggregate capabilities
  • Compute/plot anomalies
  • Composite and Difference maps
  • Compute basic statistics

    • Create Time Series Plots

  • Download Data, Maps, Plots
  • Access metadata

    • What is the Gridpoint Time Series Tool?

    An interactive web-based interface that can be used to look at time series plots of various climate elements at individual grid points.

    What dataset is being used?

    This tool works with the NRCC interpolated, NRCC Hi-Res gridded datasets developed at NRCC (Northern regional Climate Center) as well as with a variety of NARCCAP modeled datasets.

    • For more information on Acis gridded data sets NRCC interpolated, NRCC Hi-Res:
      • GridACIS grids 1 and 3
    • For more information and direct access to NARCCAP (North American Regional Climate Change Assessment Program) gridded data sets:
      • NARCCAP data page

    What is the Climate Summary Map Tool?

    An interactive web-based interface that can be used to investigate availability of data for the period of record of a station in the COOP network for the 5 most common climate elements

    • Maximum Temperature
    • Minimum Temperature
    • Total Precipitation
    • Total Snowfall
    • Total Snowdepth

    What dataset is being used?

    This tool works with COOP station data set found in the ACIS system.

    Acis Home Page

    What can you do with this tool?

    Get a graphic summary of the availablity of data of the five most common elements of a COOP station

    What is the Monthly Averages Tool?

    An interactive web-based tool that can be used to visualize monthly averages over user chosen range of years at a station location.

    What dataset is being used?

    This tool works with station data found in the ACIS system.
