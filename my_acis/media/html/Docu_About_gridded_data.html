What is this tool?

Modeled/Gridded Data access point.

What datasets are available?

NRCC Interpolated

NRCC Interpolated Hi-Res

PRISM

NARCCAP - CRCM

NARCCAP - HRM3

NARCCAP - MM5I

NARCCAP - RCM3

NARCCAP - WRFG

All data sets are made available through the ACIS system.

ACIS Home Page

What can you do with this tool?

Access and download data
Obtain data for

  • Individual grid points
  • Grid points within a county, county warning area
  • Grid points within a climate division, drainage basin
  • Grid points within a custom polygon, circle or bounding box
Available climate elements
  • Max/Min/Ave Temperatures
  • Precipitation
  • Heating/Cooling/Growing Degree Days
Summarize data temporally or spatially (mean, max, min, sum)
Available Data Formats: .xls, .dat, .txt, html(prints to screen)
