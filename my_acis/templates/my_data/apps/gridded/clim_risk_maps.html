Grid selection is made either by US state or by a user defined bounding box.

Select grid by = "State". Enter a US state of choice.

Select grid by = "Bounding Box". Find data for all stations in a bounding box of your choice.

A bounding box has four coordinates: West longitude, South latitude, East longitude, North latitude. Please enter them in this order OR better, use the map interface to define your bounding box.

Select grid by = "Shape". Draw a custom shape via the map interface.
