Elements
Choose one or more climate elements from the drop down menue.
Only a limited set of climate elements are available for gridded data.
Following climate elements area available.
  Station elements
  • Maximum Temperature
  • Minimum Temperature
  • Average Temperature
  • Observation Time Temperature
  • Precipitation
  • Snowfall
  • Snowdepth
  • Degree Days (Heating/Cooling/Growing)
    • Heating Degree Days base temperature 65F
    • Cooling Degree Days base temperature 65F
    • Growing Degree Days base temperature 50F
