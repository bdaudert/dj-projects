Data Summary
  • Windowed data:
      Get subsets of the period covered by start/end date.
      For example, if you don't want to get all data from 2000 but just
      want to get all January values from 2000 to 2010,set:
      Start Date 2000-01-01
      End Date 2009-12-31
      Start Window 01-01
      End Window 02-31
  • Temporal/Spatial Sumaries:
      Summarize the data over space or time.
      A summary over space results in time series.
      A summary over a time range results in one value at each grid point.
      Summary options are
    • Maximum: Finds the maximum value over an area or time range.
    • Minimum: Finds the mimimum value over an area or time range.
    • Mean: Finds the average value over an area or time range.
    • Sum: Finds the sum of the values over an area or time range.
