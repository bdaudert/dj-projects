Official IPython Notebook documentation

Writing your notebook

1. Start a notebook server.


    ipython notebook
2. Open this browser http://127.0.0.1:8888/

3. Start a new notebook.



4. Write your new notebook.

The interface is intuitive and easy to use.
For in-depth instructions, please consult the documentation.

Saving and storing your notebook

1. Download the notebook as .ipynb file



2. Push it to your public github repository

Sharing your notebook

Everyone that has the github link to your notebook file
can simply go to nbviewer,
enter the link and hit "Go" to view your notebook in a browser.
