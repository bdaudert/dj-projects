1. Netcdf to hdf5 conversion

Since data in the ACIS system is stored in hdf5 files, we first need to convert all data files into this format. Below is the python script that converts the MACA netCDF files into HDF5 format. Each HDF5 file (yyyy.h5) contains data for one year for all elements.

MACA_HDF5_converter.py

Describes methology for conversion of netCDF to HDF5 format. (Internal)

2. Grid Definition file

The grid definition files contains information about the data grid, the projection and how to convert between the it and the ACIS grid (WGS 84). The file also contains information about the climate elements.

MACA grid_def.py

Describes grid, elements and coordinate transformations.
